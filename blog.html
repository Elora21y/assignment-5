Question-1: What are the different ways to select an element in the DOM?

  • document.getElementById('id name');
  • document.getElementByClassName('class name');
  • document.getElementByTagName('tag name');
  • document.querySelector('selector');
  • document.querySelectorAll('selector');

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML property is used to access or change the HTML content (tags and text content) in an element. Otherwise innerText property is used to access or change text content in an element . And textContent property is used to access or change text content with hidden text in an element.

Question-3: What is event delegation in the DOM?

Event delegation is a system where parent element handles its child elements . Jekhane event hobe sekhane handle na kore tar parent element ke handles kora hoy . In use this propagation easily avoid adding many event listeners and it s work event bubbling process .

Question-4: What is event bubbling in the DOM?

Event bubbling is started from target element and its end to the root element(document) . Its starts from the child element and moves step by step towards the parent element like bubbles and reaches the root element .

Question-5: How do you create, add, and remove elements using JavaScript?

createElement method is used to create new HTML element dynamically in DOM . Example : const p = document.createElement('p'); Here new paragraph tag is created .
add() method is used of the classlist to add one or more classes to an element in DOM . And remove() method is used to remove HTML element from the DOM.
